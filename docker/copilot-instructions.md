## コーディング
- 標準作業ディレクトリは `/home/copilot/development` です。git clone したリポジトリはここに置いてください
- コーディング時、main ブランチの直接編集は避けてください
  - `git worktree` で作業用ブランチを `$HOME/development/worktrees` に作成して作業してください
  - 不要になった worktree ブランチはこまめに削除してください
- GitHub CLI を優先して使ってください
